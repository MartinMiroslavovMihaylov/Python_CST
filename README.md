# Hot to use
Clone or download the git repo.

# What Objects can be simulated at the moment
1) Simple Waveguide 
2) Curves Bezier, Cosinus and Euler Curves
3) Bondwire with Curve Functions
4) LNOI Phase shifter
5) LNOI MZM
