# Python_CST 
This is an Python CST Wraper. Please check the functionalitys of the script and capabilities under in this README file. 
For Example how to create this structures you can go to the online documentation for more information.

# Objects that can be design with the scripts
	- Brick
	- Code
	- 2D and 3D Polygon to solid 
	- Bond Wires
	- Create Curves
	- Optical waveguide
	- Create Mach-Zehnder-Modulator
	- Phase Modulator

# Operations that can be done with the script 
	- Set Units
	- Set Frequency
	- Set Wavelength
	- Add and Remove Global Variables
	- Define Background 
	- Define Boundary 
	- Translate Elements
	- Cut Elements
	- Pick face/ Clear Pick  
	- Create Bezier, Cosinus and Euler Curves
	- Set Time Solver
	- Set Frequency Solver 
	- Set Monitors 
	- Set Mesh
	- Run Time Solver
